Download Python version 3.13.0 or the closest version to that (3.13.X) if the link here doesn't work.

!!! INSTALL WITH ADMIN AND 'PATH' ENABLED !!!
  https://www.python.org/ftp/python/3.13.0/python-3.13.0-amd64.exe


Run these in terminal/command prompt/powershell (dependencies):

1) UPGRADE PIP TO THE NEWEST VERSION:

WINDOWS:
  python -m pip install --upgrade pip
or
  py -m pip install --upgrade pip

MACOS / LINUX:
  python3 -m pip install --upgrade pip
or
  sudo python3 -m pip install --upgrade pip

2) DOWNLOAD THE MODULES NEEDED:
  pip install pygame
  pip install keyboard
